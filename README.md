# 💰 Finance Manager

A personal finance tracking application for managing income and expenses with category-based statistics.

## ✨ Features

- 💵 **Income & Expense Tracking**: Add transactions by category
- 📊 **Automatic Balance Calculation**: Real-time balance updates
- 📈 **Statistics Dashboard**: Visual breakdown by category
- 📅 **Date Filtering**: Filter transactions by date range
- 🎨 **Interactive Charts**: Pie chart showing expense distribution
- 💾 **Data Persistence**: All data saved in browser localStorage
- 🗂️ **Category Management**: Organize finances by categories (Transport, Food, Clothes, etc.)

## 🚀 How to Use

1. Add income using the **Bonus** category or expenses using other categories
2. View your current balance at the top
3. Check statistics sidebar for spending breakdown
4. Use date filters to view specific periods
5. Click ❌ to remove transactions

## 🛠️ Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- localStorage API
- Chart.js / Canvas for visualizations
